Read an integer N. For all non-negative integers i < N, print i^2. See the sample for details.

Input Format:

	The first and only line contains the integer, N.

Output Format:

	Print N lines, one corresponding to each i.

Constraints:

	1 <= N <= 20

ex:

	input:
		5

	output:
		0
		1
		4
		9
		16
