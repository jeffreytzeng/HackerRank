Given a sentence, s, print each word of the sentence in a new line.

Input Format:

	A line of sentence, s.

Output Format:

	As described.

Constraints:

	1 <= len(s) <= 1000

ex1:

	input:
		This is C

	output:
		This
		is
		C

ex2:

	input:
		Learning C is fun

	output:
		Learning
		C
		is
		fun
