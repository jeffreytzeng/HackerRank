Print Hello, World! on a single line, and then print input string to the stdout.

Input Format:

	A string from stdin.

Output Format:

	Print Hello, World! on the first line, and the string from the given input on the second line.

ex:

	input:
		Welcome to C programming.

	output:
		Hello, World!
		Welcome to C programming.
