Read 3 numbers from stdin and print their sum to stdout.

Input Format:

	A single line containing 3 space-separated integers: a, b, and c.

Output Format:

	Print the sum of the three numbers on a single line.

Constraints:

	1 <= a, b, c <= 1000

ex:

	input:
		1 2 7

	output:
		10
